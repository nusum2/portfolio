## 쇼핑몰 프로젝트 - Foodmart
다양한 식품들을 판매하는 쇼핑몰을 구현한 프로젝트입니다.</br>
관리자 페이지와 사용자 페이지로 구성 되어 있으며</br> 
아마존 aws와 깃허브 actions을 활용한 자동배포 기능도 추가했습니다.
